Краткое руководство по использованию основных команд Git


1.) Чтобы создать новую ветку репозитория локально необходимо следующее:

• git branch <branch_name>

2.) Чтобы на неё переключиться нужно сделать следующее:

• git checkout <branch_name>

3.) Дальше мы можем также добавлять новые файлы в текущей ветке и фиксировать эти изменения как мы делали это прежде:

• git add . 

• git commit -m "Commit message"

4.) После всех проделанных операций с новой веткой мы можем опубликовать эти изменения удалённо (*новая ветка в удалённом репозитории создастся автоматически):

• git push --set-upstream origin <branch_name>
