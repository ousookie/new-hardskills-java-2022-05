Краткое руководство по использованию основных команд Git


1.) Сначала нужно склонировать свой удалённый репозиторий к себе на ПК:

• git clone https://github.com/Имя_Пользователя/Название_Удалённого_Репозитория.git

2.) После этого появится новая папка с названием удалённого репозитория (P.S. Это просто обычная папка и с ней можно делать что угодно)

3.) Теперь можно переносить (или копировать) туда нужные файлы

4.) Чтобы Git начал отслеживать новые файлы в этой папке нужно сделать следующее:

• Узнать путь до этой папки
• Перейти по этому пути в коммандной строке:
	•• Открыть программу cmd.exe 
	•• cd путь_до_нужной_папки
• git add Имя_Файла_Который_Был_Скопирован ИЛИ git add .

4.) Чтобы Git зафиксировал изменения репозитория нужно сделать следующее (также через cmd.exe):

• git commit -m "Сообщение об изменении" (P.S. Сообщение может быть абсолютно любое)

5.) Чтобы новые файлы появились удалённо нужно сделать следующее:

• git push
